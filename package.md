### Packageの使い方
[GoDoc](https://godoc.org/)

#### hgをインストール
```bash
# Debian/Ubuntu
$ apt-get install mercurial

# Fedora
$ yum install mercurial

# Gentoo
$ emerge mercurial

# FreeBSD
$ cd /usr/ports/devel/mercurial
$ make install

# Solaris 11 Express
$ pkg install SUNWmercurial
```

#### go get
```go
go get "code.google.com/p/go.crypto/ssh"
```
